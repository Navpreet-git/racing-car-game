Rules of the Game
  1. Each player must enter their name to join the game.
  2. To create a game, click the "Create game" button and share the game code with friends.
  3. To join a game, enter the game code and click the "Join game" button.
  4. Once all players have joined, the game creator can start the game by clicking the "PLAY" button.
  5. Use the arrow keys to control your car and race against other players.
  6. The first player to reach the finish line wins the game.
  7. Have fun and play fair!
